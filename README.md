# Space Invader Game (Python + Pygame)

A simple Space Invader game built using Python and Pygame.

## How to Play
- Use the left and right arrow keys to move your spaceship.
- Press the Spacebar to shoot bullets.
- Destroy aliens to earn points.
- The game ends when enemies reach the bottom of the screen.

## How to Run
1. Make sure you have Python 3 installed.
2. Install Pygame:
   pip install pygame
3. Run the game:
   python main.py
